Recommended Starting Point

Core Bots are the most practical first implementation: color carries provider identity, silhouette carries tier, and the shapes are simple enough to redraw inside the existing canvas renderer.

Implementation Notes

  • Provider color should be theme-tokenized, not baked into agent state.
  • Tier shape can map to model size, configured agent tier, or estimated play cost.
  • Status can layer on top later with pulses, outlines, or small floating markers.
